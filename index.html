git-vimscript-installer by travisjeffery

rakefile to install vim scripts downloaded from github and other git repos

Installing VimScript's from Git Repos sucks, this just makes it suck less.

Dependencies

Ruby

Install

Add this Rakefile to the repos of your Git VimScript projects for everyone to just have to do `rake` to update and install your project.
